# 13Ft

> This is a simple self hosted server that has a simple but powerful interface to block ads, paywalls, and other nonsense.

## Links
- [Official Repository](https://github.com/wasi-master/13ft)
- [Service Configuration](https://github.com/traefikturkey/onramp/tree/main/services-available/13ft.yml)

## Docker Images
- `ghcr.io/wasi-master/13ft:${_13FT_DOCKER_TAG:-latest}`

## Environment Variables

| Variable | Default | Description |
|----------|---------|-------------|
| `HOST_DOMAIN` |  | Host domain for service access |
| `PGID` |  | Group ID for file permissions |
| `PUID` |  | User ID for file permissions |
| `TZ` |  | Timezone setting |
| `_13FT_AUTOHEAL_ENABLED` |  | Enable Autoheal container restart on unhealthy status |
| `_13FT_CONTAINER_NAME` |  | Container name |
| `_13FT_DOCKER_TAG` |  | Docker image tag/version |
| `_13FT_HOST_NAME` |  |  13ft host name |
| `_13FT_MEM_LIMIT` |  |  13ft mem limit |
| `_13FT_RESTART` |  | Container restart policy |
| `_13FT_TRAEFIK_ENABLED` |  | Enable Traefik reverse proxy |
| `_13FT_WATCHTOWER_ENABLED` |  | Enable Watchtower auto-updates |

## Configuration

### Volumes
- `/etc/localtime:/etc/localtime` - Volume mount

### Networks
- `traefik`

### Labels
**Traefik Configuration:**
- `traefik.enable=${_13FT_TRAEFIK_ENABLED:-true}`
- `traefik.http.routers.13ft.entrypoints=websecure`
- `traefik.http.routers.13ft.rule=Host(`${_13FT_HOST_NAME:-13ft}.${HOST_DOMAIN}`)`
- `traefik.http.services.13ft.loadbalancer.server.port=5000`

**Watchtower Configuration:**
- `com.centurylinklabs.watchtower.enable=${_13FT_WATCHTOWER_ENABLED:-true}`

**Other Labels:**
- `autoheal=${_13FT_AUTOHEAL_ENABLED:-true}`
- `joyride.host.name=${_13FT_HOST_NAME:-13ft}.${HOST_DOMAIN}`

## Quick Start

```bash
# Enable the service
make enable 13ft

# Configure environment variables (if needed)
make scaffold 13ft

# Start the service
make up
```
