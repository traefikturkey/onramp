## Optional: Enable NFS Storage

To store manga on an NFS share instead of locally:

1. Enable the NFS override:
   make enable-override kaizoku-nfs

2. Ensure NFS is configured in your environment:
   make edit-env-nfs
   # Set NFS_SERVER if not already configured

3. (Optional) Set a custom NFS path for Kaizoku manga:
   make edit-env-nfs
   # Add: KAIZOKU_NFS_SERVER_PATH=/your/nfs/path
   # If not set, defaults to NFS_MEDIA_PATH

4. Restart to apply:
   make restart-service kaizoku
